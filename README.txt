Instalación:

Antes de comenzar, debemos asegurarnos de tener instalada la JVM para poder ejecutar el programa.

Una vez descargado el zip Tacos.rar, desempaquetarlo en el escritorio, mover los archivos de access (tienen las extensiones .accdb), en su directorio C y ya puede ejecutar la app.
